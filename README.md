# Site de 2 anos de casamento

Arquivos incluídos:
- `index.html`
- `style.css`
- `script.js`

## Como publicar no GitHub Pages

1. Crie um repositório no GitHub.
2. Envie estes arquivos para o repositório.
3. Vá em **Settings > Pages**.
4. Em **Build and deployment**, escolha:
   - **Source:** Deploy from a branch
   - **Branch:** main
   - **Folder:** /(root)
5. Salve.
6. Aguarde o GitHub publicar o site.
7. O link ficará parecido com:
   `https://seuusuario.github.io/nomedorepositorio/`

## Onde editar

### Nome do casal
Edite no arquivo `index.html`:
- `Wyderson Faraco`
- `Dieli Bastos`

### Data do casamento
Edite no arquivo `script.js`:
- `2024-04-23T00:00:00`

### Carta de amor
Edite a seção `Uma carta de amor` no `index.html`.

### Fotos
Substitua os blocos `Adicione uma foto aqui` por imagens com a tag `<img>`.

Exemplo:
```html
<img src="foto1.jpg" alt="Nossa foto" class="real-photo">
```
